HTML

HTML (HyperText Markup Language) é a linguagem padrão usada para criar e estruturar páginas na web. Desenvolvida no início da década de 1990 por Tim Berners-Lee, ela permite organizar conteúdos como textos, imagens, links e vídeos dentro de um site. Por meio de tags e elementos, o HTML define a estrutura básica de uma página, servindo como a base sobre a qual outras tecnologias da web funcionam.

CSS

CSS (Cascading Style Sheets) é a linguagem usada para estilizar páginas criadas com HTML. Ela permite definir cores, fontes, espaçamentos, layouts e animações, tornando os sites visualmente atraentes. Criado para separar o conteúdo da apresentação, o CSS facilita a manutenção e o design de páginas web, sendo uma das principais ferramentas para o desenvolvimento de interfaces modernas.

JavaScrypt

JavaScript é uma linguagem de programação amplamente utilizada para adicionar interatividade às páginas web. Com ela, é possível criar animações, validar formulários, atualizar conteúdos sem recarregar a página e desenvolver aplicações completas no navegador. Desde sua criação em 1995 por Brendan Eich, JavaScript se tornou uma das tecnologias mais importantes do desenvolvimento web moderno.
